= Installation
:navtitle: Installation
:description: Install Valentus Theme using the release UI bundle (chrome + bundled dark mode).

== Release UI bundle (recommended)

**Use `ui-bundle.zip` from GitHub Releases.** One pinned `ui.bundle.url`; doc-site chrome and dark mode are pre-merged.

Successor to the former `antora-dark-theme` package — migrate playbook URLs to Valentus.

* link:https://github.com/antora-supplemental/valentus-theme/releases[GitHub Releases]
* link:https://antora-supplemental.github.io/docs/[Antora Supplemental docs portal] (extensions registry)

== Steps

. Configure `antora-playbook.yml`:
+
[source,yaml]
----
ui:
  bundle:
    url: https://github.com/antora-supplemental/valentus-theme/releases/download/v1.1.0/ui-bundle.zip
    snapshot: true
----

For floating, line-pin, or Renovate strategies see xref:version-pinning.adoc[Version Pinning].

. Optional branding—*one* override directory:
+
[source,yaml]
----
  supplemental_files: ./supplemental-ui-overrides
----

. Build:
+
[source,bash]
----
antora antora-playbook.yml
----

Pin `releases/download/vX.Y.Z/ui-bundle.zip` in CI.

[[migrate-from-antora-dark-theme]]
== Migrate from `antora-dark-theme`

Replace:

[source,yaml]
----
ui:
  bundle:
    url: https://github.com/antora-supplemental/antora-dark-theme/releases/.../ui-bundle.zip
----

With:

[source,yaml]
----
ui:
  bundle:
    url: https://github.com/antora-supplemental/valentus-theme/releases/download/v1.1.0/ui-bundle.zip
    snapshot: true
----

Remove `antora-dark-theme` from `package.json` if present; use the release bundle URL or `pnpm add -D valentus-theme` for supplemental vendoring.

[[overrides-on-bundle]]
=== Overrides

Logo, favicon, or a few partials in a single directory (for example `supplemental-ui-overrides/`).
See xref:configuration.adoc#virtual-supplemental-files[virtual supplemental files] for single-file entries.

== Vendor `supplemental-ui` (forks)

Copy `supplemental-ui/` from this repository into git for a maintained fork.
Run `pnpm dark-mode:sync` to refresh vendored dark mode CSS/JS from link:https://github.com/antora-supplemental/antora-dark-mode[antora-dark-mode].

== Verify

Open the built site and use the sun/moon navbar toggle; preference should persist across pages.
