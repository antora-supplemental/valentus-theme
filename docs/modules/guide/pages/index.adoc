= Valentus Theme
:navtitle: Getting Started
:description: Composed Antora doc-site chrome with bundled dark mode — mast, header, layout, and VCS icons.

Valentus is debranded doc-site chrome for Antora 3.x: sticky header, breadcrumb mast with component/version pickers, sidebar layout, VCS provider icons, and the dark mode toggle.

The release `ui-bundle.zip` includes dark mode (successor to the former `antora-dark-theme` package).

== Install

See xref:installation.adoc[Installation] and xref:ui-modules.adoc[UI Modules].

[source,yaml]
----
ui:
  bundle:
    url: https://github.com/antora-supplemental/valentus-theme/releases/download/v1/ui-bundle.zip
    snapshot: true
----
