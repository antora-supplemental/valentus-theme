== [1.1.1] - 2026-06-13

* *Dark-mode toggle:* Default `site.keys.dark_mode_navbar` to `light` so the theme toggle uses a dark icon on Valentus' light top bar; vendors `antora-dark-mode` v1.3.1 navbar-tone support.

== [1.1.0] - 2026-06-13

* *Bundled dark mode:* Release `ui-bundle.zip` merges link:https://github.com/antora-supplemental/antora-dark-mode[antora-dark-mode] with Valentus chrome. Successor install path for former `antora-dark-theme` consumers.
* *Partials:* `head-meta.hbs` and `footer-scripts.hbs` include FOUC prevention, `site-extra.css`, and `site-dark-mode.js`.
* *Maintainers:* `pnpm dark-mode:sync` vendors dark mode CSS/JS from `antora-dark-mode`; release CI runs sync before bundling.

== [1.0.0] - 2026-06-09

* Initial release: doc-site chrome split from `antora-dark-theme` into `valentus-theme` (mast, header, layout, VCS icons, UI modules orchestrator).
* Pair with link:https://github.com/antora-supplemental/antora-dark-mode[antora-dark-mode] for dark styling and the theme toggle.
* GitHub Pages demo via `.github/workflows/deploy.yml`.
